# AI Created Games

Claude Code（AIコーディングエージェント）で作成したブラウザゲーム集です。

以下の書籍のハンズオンをベースに開発しました。
https://github.com/GenerativeAgents/claude-code-book

## ゲーム一覧

### ブロック崩し
- 10×5のブロックをボールで壊していくクラシックゲーム
- **アイテムシステム**: 20個壊すごとにランダムアイテムが出現
  - ×5マルチボール / スピード×2 / 5秒間無敵（連続破壊）
- クリア時にくす玉・紙吹雪の演出あり
- ポーズ機能（⏸ ボタン / P キー）

### テトリス
- Sega Tetris（1988年アーケード版）仕様に準拠
- スコア: 1列=100 / 2列=400 / 3列=900 / 4列=2000
- 4ライン消去ごとにレベルアップ、落下速度が上昇
- コロベイニキBGM（Web Audio API）、ハイスコア記録（上位5件）
- ポーズ機能（⏸ ボタン / P キー）

### 4目並べ（Connect4）
- 縦・横・斜めに4つ並べると勝ち
- CPUまたは2人対戦に対応
- CPU難易度: かんたん / ふつう / むずかしい / とてもむずかしい（置換表付きminimax）
- 待った機能（3回まで）、ゲームオーバー後も使用可能
- 駒の落下アニメーション、勝敗オーバーレイ表示

## 動作環境

- モダンブラウザ（Chrome / Firefox / Safari）
- スマートフォン対応（タッチ操作）

## 起動方法

```bash
npx http-server . -p 8080 --cors -c-1
```

ブラウザで http://localhost:8080 を開いてください。
